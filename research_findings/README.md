# Research Findings

This directory contains research reports and analysis conducted by the @researcher agent.

## Purpose

The @researcher agent is a specialized AI assistant that conducts deep research, web scraping, and data analysis. When invoked with `@researcher`, it can:

- Search the web for current information
- Interact with websites using Playwright and Hyperbrowser
- Analyze and synthesize information from multiple sources
- Create comprehensive research reports
- Commit findings directly to this directory

## File Naming Convention

Research reports are named using the format:
```
YYYY-MM-DD-topic-name.md
```

For example:
- `2025-11-14-next-js-state-management.md`
- `2025-11-14-competitor-analysis.md`

## Report Structure

Each research report typically includes:

1. **Executive Summary** - Quick overview of findings
2. **Research Objectives** - What questions were being answered
3. **Methodology** - How the research was conducted
4. **Findings** - Detailed results organized by category
5. **Analysis** - Interpretation of the findings
6. **Recommendations** - Actionable next steps
7. **Sources** - All references and citations

## How to Request Research

To request research from the @researcher agent:

1. Create an issue or comment mentioning `@researcher`
2. Clearly state your research question or objective
3. Specify any particular sources or constraints
4. The researcher will conduct the investigation and commit findings here

## Examples of Research Tasks

- Technology comparisons and evaluations
- API documentation and integration research
- Competitor analysis
- Best practices research
- Security vulnerability research
- Market analysis
- Documentation gathering
